Intro

Hi! I'm Ashton, a full-stack web developer living in Charlotte, North Carolna.

I'm a tech enthusiast who embarked on my coding journey on my own practicing javascript on nights and weekends while holding a full time job. I leveled up my skills while attending and graduating from Fullstack Academy in New York. I found my groove in the tech world by specializing in automation testing, laying the groundwork for a dynamic career in React Native development.

My early focus on automation testing equipped me with a sharp eye for spotting and resolving issues in the software development pipeline. Crafting automated test scripts using webdriverIO to write end to end tests became second nature, ensuring the reliability and functionality of Northwestern Mutuals client website.

Transitioning into React Native development felt like a natural progression. Drawing on my automation testing background, I brought a unique perspective to the table, emphasizing efficiency, scalability, and a user-centric approach in mobile app development.

In the realm of React Native, I've collaborated seamlessly with designers and cross-functional teams, contributing to the delivery of polished and responsive mobile application used by Financial Representatives to track clients and appointments. My automation testing roots have influenced my development approach, prioritizing resilience and high performance in the software I create.

My story, from automation testing to React Native development, showcases the transformative power of coding bootcamps and the potential for growth within the tech industry. With an innovative mindset and a commitment to excellence, I'm excited about making a lasting impact in the world of React Native and beyond.
